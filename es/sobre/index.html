INICIO SOBRE PROYECTOS BLOG

Sobre mí

Ingeniero de Software Backend centrado en automatización, integraciones de APIs, y sistemas escalables. Diseño servicios backend con Python, Node.js, SQL y APIs de nube para conectar plataformas empresariales fragmentadas en flujos de trabajo operativos confiables.

Mi trabajo reciente incluye plataformas backend que soportan 8K-12K transacciones diarias, flujos de automatización que eliminaron ~3 horas/día de informes manuales por analista, pipelines ETL que reducen el tiempo de procesamiento en ~40%, y optimizaciones API que mejoraron los tiempos de respuesta al 35-40%.

Trabajo en arquitectura, implementación y gobernanza de sistemas para convertir los cuellos de botella de las finanzas, las operaciones y los informes en sistemas backend mensurables con uso en el mundo real.

Experiencia profesional y entornos empresariales

Loja del Sapo

Arquitectura backend, automatizaciones, integraciones y sistemas operativos.

3 roles
  • Diseñé sistemas backend internos con Python y PostgreSQL para admitir transacciones diarias de 8K a 12K en flujos de trabajo operativos.
  • Diseñé las automatizaciones impulsadas por GitHub Actions y API que eliminaron ~3 horas al día de informes financieros manuales por analista.
  • Construimos pipelines ETL entre Google Cloud, Zoho y servicios de terceros, lo que redujo el tiempo de procesamiento de datos en aproximadamente un 40 %.
  • Se entregaron paneles de control en tiempo real, filtros personalizados y estándares de gobierno backend para múltiples sistemas internos.
  • Desarrollé integraciones de REST en Python y Node.js en más de 6 sistemas empresariales centrales que abarcan Google Cloud, Zoho y servicios externos.
  • Flujos de trabajo automatizados de extracción y enriquecimiento de datos, lo que reduce la entrada manual y mejora la precisión de los datos en los sistemas operativos.
  • Se aplicó almacenamiento en caché y procesamiento asíncrono a los servicios backend, mejorando los tiempos de respuesta de API entre un 35 y un 40 %.
  • Mantuve los procesos AWS S3 y Lambda mientras aplicaba las reglas de backend para descuentos, validaciones y restricciones operativas.
  • Implementé más de 20 flujos de trabajo automatizados y reglas comerciales con Python y Deluge para reducir el trabajo operativo repetitivo.
  • Creó pipelines de informes y estructuras de datos para el seguimiento operativo en tiempo real y la visibilidad del cumplimiento.
  • Desarrollé flujos de procesamiento validados que mejoraron la calidad de los informes y la coherencia de la ejecución.
