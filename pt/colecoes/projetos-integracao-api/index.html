INÍCIO SOBRE PROJETOS BLOG

Projetos de Integração de APIs e Coleções de Sistemas

Esta página é um cluster direto para descoberta do sistema de integração de APIs. Ele agrupa todas as páginas do projeto relacionadas à integração e as organiza por função: orquestração, extração de provedor, ERP sincronização e confiabilidade orientada a eventos.

Camada de Orquestração

  • Google Auth Worker: geração de token, atualização segura de segredo, envio de repositório.

Workers de provedores e ETL

  • Pipeline de ingestão bruta de APIs: ingestão bruta de APIs em tabelas brutas Supabase SQL para dados Hablla, Zoho Creator, Zenvia Voice e SIGE ERP.
  • Zoho Integration Worker: Zoho Extração do criador, mapeamento, sincronização.
  • Hablla Integration Worker: pipeline de agregação de cartões e atendimento.
  • Zenvia Integration Worker: relatando ingestão e filtragem.

ERP Pipelines de integração

  • SIGE Integration Worker: transformação de faturamento e reconciliação.
  • Omie Integration Worker: vendas e sincronização de produtos.

Confiabilidade de eventos e backend

  • Event-Driven Integration Service: processamento de webhook e arquitetura orientada a eventos.
  • Multi-Tenant SaaS Platform: arquitetura backend de APIs e processamento de trabalhos.

Contexto da coleção

Esses projetos estão conectados a responsabilidades reais de integração documentadas em currículos e notas de trabalho: integração em mais de 6 sistemas principais, API primeira troca em mais de 4 plataformas, depuração de endpoint e resultados de automação operacional vinculados a fluxos de trabalho de relatórios.
